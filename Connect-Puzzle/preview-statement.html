Connect Puzzle

各成分が正の整数である \(N\) 行 \(N\) 列の行列 \(A = A_{i,j}\) であって次の条件を満たすものが存在するか判定し、存在する場合は \(\max_{1 \leq i, j \leq N}(A_{i,j})\) が最小となるものを1つ出力してください。

  • 行列 \(A\) をトーラス状とした時、どの縦・横・斜めを見ても整数 \(k\) は \(k\) 個以下しか連続していない。

制約

  • \(1 \leq N \leq 1000\)
  • 入力は整数

入力

入力は以下の形式で標準入力から与えられます。

\(N\)

出力

条件を満たす行列が存在しない場合、No を出力してください。

条件を満たす行列が存在する場合、\(\max_{1 \leq i, j \leq N}(A_{i,j})\) が最小となるものを以下の形式で出力してください。

Yes
\(A_{1,1}\; A_{1,2}\; \ldots A_{1,N}\)
\(A_{2,1}\; A_{2,2}\; \ldots A_{2,N}\)
\(\vdots\)
\(A_{N,1}\; A_{N,2}\; \ldots A_{N,N}\)

入力例1

4

出力例1

Yes
1 2 3 2
3 3 2 3
2 3 2 1
3 3 3 2

入力例2

1

出力例2

No
