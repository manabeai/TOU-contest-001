## 問題文
大学は、単位を順調に習得していくと $4$ 年後に卒業となります。
$N$ 年に入学した学生は、何年に卒業するか出力してください。

## 制約

- $2000 \leqq N \leqq 3000$
- 入力はすべて整数

## 入力
入力は以下の形式で標準入力から与えらる。  
```md
$N$

```
## 出力
答えを整数として出力せよ。

## サンプル1

### 入力
```
2000

```

### 出力
```
2004

```
$2000$ 年の $4$ 年後は、$2000 + 4 = 2004$ のため $2004$ 年です。

## サンプル2

### 入力
```
2022

```

### 出力
```
2026
```
