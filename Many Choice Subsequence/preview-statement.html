Many Choice Subsequence

正整数 \(N\) および長さ \(N\) の整数列 \(A,B,C\) が与えられます。

\(1 \leq l \leq r \leq N\) なる整数の組 \(l,r\) について、以下の条件を満たすとき、区間 \([l,r]\) を良い区間であると呼びます。

  • \(\sum_{k=l}^{r}B_k \leq \sum_{k=l}^{r}A_k\)

また、良い区間 \([l,r]\) のスコアを

  • \(\sum_{k=l}^{r}C_k\)

で定義します。

区間に重複が無いように良い区間を \(0\) 個以上選んだとき、スコアの合計としてあり得る最大値を求めてください。

より厳密には、 \(1 \leq l_1 \leq r_1 < l_2 \leq r_2 < \dots < l_m \leq r_m \leq N\) なる \(m\) 個の整数の組であって、 \(1 \leq j \leq m\) を満たす全ての \(j\) に対し
\(\sum_{k=l_j}^{r_j}B_k \leq \sum_{k=l_j}^{r_j}A_k\) を満たすものについて \(\sum_{j=1}^{m}\sum_{k=l_j}^{r_j}C_k\) の最大値を求めてください。

制約

  • \(1 \leq N \leq 2 \times 10^5\)
  • \(0 \leq A_i \leq 2 \times 10^5\)
  • \(0 \leq B_i \leq 2 \times 10^5\)
  • \(0 \leq \sum_{k=1}^{N}A_k \leq 2 \times 10^5\)
  • \(0 \leq \sum_{k=1}^{N}B_k \leq 2 \times 10^5\)
  • \(1 \leq C_i \leq 10^9\)
  • 入力はすべて整数

入力

入力は以下の形式で標準入力から与えられます。

\(N\)
\(A_1\) \(A_2\) \(\dots\) \(A_N\)
\(B_1\) \(B_2\) \(\dots\) \(B_N\)
\(C_1\) \(C_2\) \(\dots\) \(C_N\)

出力

答えを \(1\) 行で出力してください。

サンプル1

入力

6
1 4 4 2 0 9
3 3 3 3 5 7
1 2 3 4 5 6

出力

15

良い区間として \(2\) つの区間 \(([2,4],[6,6]\) を選んだ場合、スコアの合計は \((2+3+4) + 6 = 15\) となり、これが最大です。

サンプル2

入力

5
0 1 2 3 4
4 3 2 1 0
1000000000 1000000000 1000000000 1000000000 1000000000

出力

5000000000

良い区間は \(\sum_{k=l}^{r}A_k = \sum_{k=l}^{r}B_k\) でも良い事に注意してください。

また、答えは \(32\) bit整数型に収まらない場合もあります。

サンプル3

入力

2
0 4
2 6
20 25

出力

0

良い区間が存在しない事もあります。
