## 問題文

2 つの整数 $a$ と $b$ が標準入力から与えられます。これらを足し合わせた結果を出力してください。

## 制約

- $1 \leq a, b \leq 10^18$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられます。  
$a$ $b$

## 出力

$a$ と $b$ を足し合わせた結果を出力してください。

## サンプル1

### 入力
```
2 3

```

### 出力
```
5

```

## サンプル2

### 入力
```
1000000000 1000000000

```

### 出力
```
2000000000

```
